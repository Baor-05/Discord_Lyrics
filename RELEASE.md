# Release Notes - DiscordLyrics v1.4.0

Chào mừng đến với phiên bản **v1.4.0** của DiscordLyrics! Bản phát hành này bổ sung nhiều tính năng mới thú vị nhằm cải thiện trải nghiệm nghe nhạc và điều khiển trạng thái Discord của bạn.

## Các tính năng mới nổi bật

1. **Bộ điều khiển nhạc 5 nút đầy đủ**:
   - Thêm các nút: **Trộn bài (Shuffle)**, **Lùi bài (Previous)**, **Phát / Tạm dừng (Play/Pause)**, **Tiến bài (Next)**, và **Lặp lại (Repeat)**.
   - Hỗ trợ đầy đủ cho cả Spotify (Local/Web) và YouTube Music (Desktop App API / Web Extension).
2. **Kéo/click thanh thời gian để tua bài**:
   - Bạn có thể nhấp chuột hoặc kéo thả trực tiếp trên thanh tiến trình để tua nhanh/chậm bài hát tới bất kỳ đoạn nào mong muốn.
3. **Tự động lưu và khôi phục kích thước cửa sổ**:
   - Ứng dụng sẽ tự động ghi nhớ vị trí và kích thước cửa sổ của bạn khi đóng, và khôi phục chính xác trạng thái đó khi bạn mở lại.
4. **Hệ thống Kiểm tra & Tự động cập nhật**:
   - Thêm tuỳ chọn tự động kiểm tra bản cập nhật mới từ GitHub Releases. Bạn có thể cập nhật nhanh chóng chỉ bằng một nút nhấn trong app.

## File đính kèm phát hành (Release Assets)

Để nâng cấp hoặc cài đặt mới, bạn cần tải về:
- **`DiscordLyricsSetup.exe`**: Bộ cài đặt tự động dành cho hệ điều hành Windows.

## Hướng dẫn cài đặt

1. Tải tệp `DiscordLyricsSetup.exe` từ phần Assets bên dưới.
2. Chạy tệp tin để cài đặt/cập nhật ứng dụng lên phiên bản mới nhất.
3. Cửa sổ ứng dụng sẽ tự động mở sau khi hoàn thành.
